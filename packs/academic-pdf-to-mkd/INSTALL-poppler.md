# Poppler Probe

`poppler` is a tier-3 system dependency because it is GPL-licensed and should not be bundled into this pack.

The v0.3 pack only probes these binaries:

- `pdftotext`
- `pdfinfo`
- `pdftoppm`

Do not install Poppler from this pack. Mode-2 install execution is a Bridle v0.4 concern.
